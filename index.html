Git

  • # In which branch are we?:
    git status
  • # Open the directory in VS Code if it's not currently open:
    code .
  • # Create a new branch:
    git checkout -b feature/starter-code
    git status
  • # Adds modifications in the current working branch to the staging area:
    git add -A
  • # Create a commit that captures a snapshot of the currently staged changes.
    # Once committed, feature/starter-code will be updated with the starter code.
    git commit -m "add starter code"
  • # Pull down any changes from the base branch:
    git pull origin main
  • # Push our changes to the remote branch:
    git push origin feature/starter-code
